# 2019nCoV疫情地图

采用头条接口，制作分布地图点标记的轨迹和地区风险等级。

uniAPP文档：https://uniapp.dcloud.io/

## 安装

```
npm install
# or
yarn 
```

## 启动项目
```
# 百度小程序
npm run dev:mp-baidu

# 头条小程序
npm run dev:mp-toutiao

# 微信小程序
npm run dev:mp-weixin
```
